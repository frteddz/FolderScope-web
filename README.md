# FolderScope

> Disk usage visualizer — see where your storage went with interactive charts.

This repository contains only the built distribution files. The source code is not publicly available.

## Features

- Scan any folder and visualize its contents by size
- Interactive treemap and sunburst chart views
- Drill down into subfolders with a single click
- Sort files and folders by size, name, or type
- Filter by file extension (e.g., show only images)
- Identify duplicate files occupying space
- Export scan results as a CSV report
- Real-time scanning progress indicator
- Full offline capability — no uploads, no server

## Why FolderScope?

FolderScope processes everything locally using the File System Access API. Your folder contents are never transmitted anywhere — the scan, analysis, and visualization all happen in your browser. It's free, fast, and gives you complete visibility into your storage without privacy trade-offs.
